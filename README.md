# CAENDT5740_DAQ
A data acquisition software for CAEN DT5740 digitizer


*Software Requirements*
- Windows comupter (This has not been tested on any other operating systems)
- Python 3.6 or above
- Python packages:
    - Numpy
    - Mathplotlib
    - caen_felib with DIG 1 (`pip install caen-felib`)
